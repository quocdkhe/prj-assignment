# Hướng Dẫn Chạy Dự Án

## Các bước để chạy dự án:

### Bước 1:
Chạy file **DBScript** để khởi tạo cơ sở dữ liệu.

### Bước 2:
Chạy file **trigger** để thêm các trigger cần thiết.

### Bước 3:
Mở project bằng **NetBeans** và bắt đầu chạy dự án.
